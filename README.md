# Inverted_pendulum_project

jupyter_project : sujet du BE avec la correction et l'animation qui doit être dans le même dossier que les notebooks.

proj : Tous mes fichiers python, avec :
  BE_script <=> script du BE en format python
  NXTwaysim <=> Paramètres du robot et fonctions nécessaires
  animation <=> Animation infinie utilisée par l'interface
  interface <=> Première interface en utilisant des Qt, ce qui est impossible sur les PCs de l'INSA
  interface_tkinter <=> Migration de l'interface en utilisant tkinter, possible de l'utiliser sur les PCs de l'INSA
  my_animation <=> animation de base sur un temps défini, utilisé dans le sujet du BE, pour une certaine entrée et une certaine dynamique
